# yup-basename

```
NAME:
   basename - strip directory and suffix from filenames

USAGE:
   basename NAME [SUFFIX]
      basename OPTION... NAME...

      Print NAME with any leading directory components removed.
      If specified, also remove a trailing SUFFIX.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --suffix value, -s value  remove a trailing SUFFIX; implies -a
   --multiple, -a            support multiple arguments and treat each as a NAME (default: false)
   --zero, -z                end each output line with NUL, not newline (default: false)
   --help, -h                show help
```
